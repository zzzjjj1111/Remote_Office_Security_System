### 验证4：数据库记录验证

**查询误判反馈记录：**

```sql
USE zero_trust_office4;

-- 查看所有误判反馈
SELECT 
    a.id,
    a.alert_level,
    a.is_false_positive,
    a.feedback_type,
    a.feedback_reason,
    a.feedback_time,
    a.baseline_corrected,
    a.correction_time,
    u.name as user_name
FROM alerts a
LEFT JOIN users u ON a.user_id = u.id
WHERE a.is_false_positive = 1
ORDER BY a.feedback_time DESC;
```

**查询误判样本记录：**

```sql
-- 查看误判样本表
SELECT 
    fps.id,
    fps.alert_id,
    fps.user_id,
    u.name as user_name,
    fps.behavior_type,
    fps.anomaly_score,
    fps.feedback_reason,
    fps.used_for_training,
    fps.created_at
FROM false_positive_samples fps
LEFT JOIN users u ON fps.user_id = u.id
ORDER BY fps.created_at DESC;
```

**验证基线修正：**

```sql
-- 查看基线配置是否更新
SELECT 
    id,
    trust_threshold,
    wma_weight,
    baseline_data,
    updated_at
FROM algorithm_config;
```

##  常见问题排查

### 问题1：数据库迁移失败

**症状：** 执行SQL时报错
**原因：** 数据库名不正确或表不存在
**解决：**
```bash
# 检查数据库名
mysql -u root -p -e "SHOW DATABASES;"

# 如果数据库名不同，修改migrate_false_positive.sql中的USE语句
```

### 问题2：API返回401未授权

**症状：** 所有API调用返回401
**原因：** Token未设置或过期
**解决：**
1. 重新登录获取新Token
2. 检查localStorage中token是否存在
3. 在API请求中正确设置Authorization头

### 问题3：前端对话框不弹出

**症状：** 点击"标记误报"无反应
**原因：** 前端代码未正确加载
**解决：**
1. 检查浏览器控制台是否有错误
2. 确认Vue组件正确导入
3. 刷新页面或重启前端服务

### 问题4：基线修正未生效

**症状：** feedback提交成功但baseline_corrected=false
**原因：** 基线数据不存在或用户无基线记录
**解决：**
1. 检查algorithm_config表是否有数据
2. 确认用户有历史行为日志
3. 查看后端日志是否有错误信息

### 问题5：批量标记失败

**症状：** 批量标记部分成功部分失败
**原因：** 部分告警ID不存在或网络问题
**解决：**
1. 检查浏览器控制台错误
2. 查看后端日志确认哪些请求失败
3. 验证告警ID是否有效
